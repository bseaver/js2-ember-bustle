## Prerequisites

You will need the following things properly installed on your computer.

* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (with NPM)
* [Bower](https://bower.io/)
* [Ember CLI](https://ember-cli.com/)
* [PhantomJS](http://phantomjs.org/)

## Installation

* `git clone <repository-url>` this repository
* `cd bustle`
* `npm install`
* `bower install`

## to create project

* ember new bustle
* npm install
* npm install -g ember-cli
* ember g route
* ember g route article
* ember g model article
* ember install emberfire
* create project in Firebase
* modify environment.js base on firebase configuration.
* add attributes to model (models/rental.js)
* create articles.json
* add findAll('article') in index.js
* ember install ember-bootstrap (uninstall: npm uninstall ember-bootstrap --save-dev)
* ember g component article-detail
